# projeto Anna-Bella-modelo. Este é um projeto simples desenvolvido como atividade de um curso, com o objetivo de praticar estruturação básica de páginas web utilizando HTML e CSS.
O projeto apresenta um modelo de site estático inspirado no tema “Anna Bella”, servindo como exercício introdutório para organização de conteúdo, estilização e boas práticas de marcação.
